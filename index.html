How to start Coding


For Kids from 4-10 or weak learners

If you are a kid of 4-10 years then you should start from using Code.org even if you have a slow learning speed at the age of 16 then also you can use Code.org . Code.org is a children friendly online tool from which you can start basic block coding for enhancing your code logic that is very important to learn big codes further ahead . When you will enter the Code.org site you will get many option for starting block coding first of all you will have to sign up to code.org . You can watch this video to know how to sign up to code.org and then which ever project you will open you will get side by side tutorials on the same web tool . The most good part is this is completely free! . I have been personally using code.org in my starting periods of coding . You should practise and learn what each code block for atleast 2-3 months . After you have done using code.org there is one more web tool same of block coding that you will have to use to enhance your coding skills further . That is Thunkable . It is also a block coding platform but a bit more difficult and complex then code.org . You might get problem finding out but there are always youtube tutorials that may help you out . There are also starter tutorials in the web tool that you can get help from. In thunkable also you will have to sign up , you may watch this video to get help how to sign in to thunkable . Thunkable will help you built the logic that is needed for coding and help you a bit better when you start doing text coding .
